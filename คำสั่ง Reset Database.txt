หากคุณต้องการลบข้อมูลทั้งหมดในตารางเพื่อเริ่มเก็บข้อมูลใหม่ (Reset) สามารถทำได้ 2 วิธีครับ:

  วิธีที่ 1: ใช้ SQL Editor (แนะนำ - สะดวกและรวดเร็ว)
  ไปที่หน้า SQL Editor ใน Supabase แล้วรันคำสั่งนี้ครับ:

 truncate table sensor_logs;

 truncate table ai_insights;
  หมายเหตุ: คำสั่ง truncate จะลบข้อมูลทั้งหมดทิ้งทันทีและทำให้ ID เริ่มนับใหม่ครับ

  ---

  วิธีที่ 2: ลบผ่านหน้า Table Editor (UI)
   1. ไปที่เมนู Table Editor (รูปตารางด้านซ้าย)
   2. เลือกตาราง sensor_logs
   3. กดปุ่ม Actions หรือเลือกแถวทั้งหมดแล้วกด Delete ครับ

  ---

  💡 ทิปเพิ่มเติม:
  ถ้าคุณอยากลบข้อมูลผ่านหน้า Dashboard (สร้างปุ่ม "Clear Data" บนเว็บ) บอกผมได้นะครับ! ผมสามารถเพิ่มปุ่มนี้ให้คุณได้ เพื่อความสะดวกเวลาเอาไปสาธิต (Demo)
  ให้อาารย์ดูครับ จะได้เคลียร์ข้อมูลโชว์ได้สดๆ เลย!
